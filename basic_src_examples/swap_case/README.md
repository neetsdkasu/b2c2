# swap_case

文字列の大文字小文字を入れ替える  


| BASICソース   | 変換後CASL2 | 概要                        |
|:--------------|:------------|:----------------------------|
| swapcase.bas  | MAIN.cas    | メイン部                    |


変換  
```
cargo run -- -src swapcase.bas
```

実行  
 - MAIN.casをプログラムのスタートとして実行する


実行例  
```
Swapcase Alphabet
? Hello,World!
hELLO,wORLD!
```
